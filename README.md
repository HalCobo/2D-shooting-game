# 2D-shooting-game

スマホ・タブレット向けに操作しやすい 2D シューティングゲームです。

## 遊び方

1. `index.html` をブラウザで開く（または静的サーバーで配信）
2. 「ゲーム開始」をタップ
3. 左の仮想スティックで移動（攻撃は常時オート連射）
4. 画面下に残機がハート表示されます。
5. 敵は右側で旋回しながら弾を撃ってきます。一定時間で消滅する際に弾を全方位へ拡散します。
6. 通常敵は2サイズあり、大型はやや硬く、撃破時の撃破カウントが2体分進みます。
7. 自機の攻撃で敵を一定数倒すと、画面右外からボスが出現演出つきで登場し、それ以降は通常敵は追加スポーンしません。
8. ボス通常攻撃は3way弾で、特殊攻撃の前後には短い隙があります。
9. 特殊攻撃は「多方向ビーム」「低速の特殊弾（定期拡散）」「取り巻き1体召喚（緑/橙/黄）」の等確率抽選です。
10. 取り巻きは同色の重複召喚がなく、緑=移動しつつ低頻度・高回復＋回復エフェクト、橙=自機狙いの長い直線ビーム、黄=移動しつつ大きめの時間差追尾弾（画面外で消滅）を担当します。
11. 敵・ボスは右画面外から侵入し、通常敵は侵入後スムーズに周回へ移行します。
12. ボスの移動範囲は画面右側エリアに制限されています。
13. ボス撃破時は出現演出と同系の「ボス撃破！」バナー表示後に、クリアタイム付きのCLEAR画面へ切り替わります。
14. 敵本体または敵弾に当たると自機が点滅し、HP が減ります（0でゲームオーバー）。

## ローカル起動例

```bash
python3 -m http.server 8000
# http://localhost:8000 にアクセス
```
