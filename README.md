# 2D-shooting-game

スマホ・タブレット向けに操作しやすい 2D シューティングゲームです。

## 遊び方

1. `index.html` をブラウザで開く（または静的サーバーで配信）
2. 「ゲーム開始」をタップ
3. 左の仮想スティックで移動、右の `FIRE` ボタンで連射
4. 敵は右側で旋回しながら弾を撃ってきます。一定時間で消滅する際に弾を全方位へ拡散します。
5. 大きい敵ほど体力が高く、以前より全体的に倒しにくくなっています。
6. 自機の攻撃で敵を一定数倒すとボス個体が登場し、それ以降は通常敵は追加スポーンしません。
7. ボスは3way弾、停止ビーム連射、接近時の分裂拡散弾など固有攻撃を行います。
8. 敵本体または敵弾に当たると自機が点滅し、HP が減ります（0でゲームオーバー）。

## ローカル起動例

```bash
python3 -m http.server 8000
# http://localhost:8000 にアクセス
```
