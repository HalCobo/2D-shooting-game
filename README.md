# 2D-shooting-game

スマホ・タブレット向けに操作しやすい 2D シューティングゲームです。

## 遊び方

1. `index.html` をブラウザで開く（または静的サーバーで配信）
2. 「ゲーム開始」をタップ
3. 左の仮想スティックで移動、右の `FIRE` ボタンで連射
4. 敵は右側で旋回しながら弾を撃ってきます。一定時間で消滅する際に弾を全方位へ拡散します。
5. 敵本体または敵弾に当たると HP が減り、0 でゲームオーバー

## ローカル起動例

```bash
python3 -m http.server 8000
# http://localhost:8000 にアクセス
```
