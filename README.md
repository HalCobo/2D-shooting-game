# 2D-shooting-game

スマホ・タブレット向けに操作しやすい 2D シューティングゲームです。

## 遊び方

1. `index.html` をブラウザで開く（または静的サーバーで配信）
2. 「ゲーム開始」をタップ
3. 左の仮想スティックで移動（攻撃は常時オート連射）
4. 敵は右側で旋回しながら弾を撃ってきます。一定時間で消滅する際に弾を全方位へ拡散します。
5. 通常敵は2サイズあり、大型はやや硬く、撃破時の撃破カウントが2体分進みます。
6. 自機の攻撃で敵を一定数倒すとボス個体が登場し、それ以降は通常敵は追加スポーンしません。
7. ボス通常攻撃は3way弾で、特殊攻撃の前後には短い隙があります。
8. 特殊攻撃は「多方向ビーム」「低速の特殊弾（定期拡散）」「取り巻き1体召喚（緑/橙/黄）」の抽選です。
9. 取り巻きは同色の重複召喚がなく、緑=回復、橙=固定ビーム、黄=時間差弾を担当します。
10. 敵は以前より攻撃間隔が少し長く、時間消滅まで少し長く滞在します。
11. 敵本体または敵弾に当たると自機が点滅し、HP が減ります（0でゲームオーバー）。

## ローカル起動例

```bash
python3 -m http.server 8000
# http://localhost:8000 にアクセス
```
