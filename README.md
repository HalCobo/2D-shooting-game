# 2D-shooting-game

スマホ・タブレット向けに操作しやすい 2D シューティングゲームです。

## 遊び方

1. `index.html` をブラウザで開く（または静的サーバーで配信）
2. 「ゲーム開始」をタップ
3. 左の仮想スティックで移動（攻撃は常時オート連射）
4. 敵は右側で旋回しながら弾を撃ってきます。一定時間で消滅する際に弾を全方位へ拡散します。
5. 通常敵は2サイズあり、大型はやや硬く、撃破時の撃破カウントが2体分進みます。
6. 自機の攻撃で敵を一定数倒すと、画面右外からボスが出現演出つきで登場し、それ以降は通常敵は追加スポーンしません。
7. ボス通常攻撃は3way弾で、特殊攻撃の前後には短い隙があります。
8. 特殊攻撃は「多方向ビーム」「低速の特殊弾（定期拡散）」「取り巻き1体召喚（緑/橙/黄）」の抽選です。
9. 取り巻きは同色の重複召喚がなく、緑=移動しつつ低頻度・高回復＋回復エフェクト、橙=固定ビーム、黄=移動しつつ大きめの時間差追尾弾（画面外で消滅）を担当します。
10. 敵・ボスは右画面外から侵入し、画面内で少し間を置いてから攻撃を開始します。
11. ボスの移動範囲は画面右側エリアに制限されています。
12. ボス撃破時は「ボス撃破！」演出の後、クリアタイム付きのCLEAR画面が表示されます。
13. 敵本体または敵弾に当たると自機が点滅し、HP が減ります（0でゲームオーバー）。

## ローカル起動例

```bash
python3 -m http.server 8000
# http://localhost:8000 にアクセス
```
